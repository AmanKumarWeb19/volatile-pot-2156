<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```
git clone https://github.com/gaurav62472744/halting-title-3346.git
```

<p>2. Install NPM packages</p>

```
npm install
```

<p>3. View on browser at localhost:3000</p>

```
npm start
```
